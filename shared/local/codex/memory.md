## Duplicate Search Checklist

Before creating issues, branches, PRs, Sentry resolutions, or board cards, search:

- issue URL
- issue number
- normalized title slug
- Sentry short ID or route
- open PR titles and bodies
- local and remote branch names
- git worktree list
- project board item URL
