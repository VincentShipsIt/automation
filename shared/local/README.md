# Shared Local Templates

Reusable prompt bodies for loops that run with local repository access.

## Folders

- `codex/` - Codex app automation prompt bodies.
- `claude/` - Claude Desktop scheduled task and local `/loop` prompt bodies.

Use local templates for source edits, tests, builds, branch/PR work, local
validation, local worktree cleanup, and any task that needs `[REPO_PATH]`.
